Calculator
This is a simple calculator program built using Python and the Tkinter GUI library.

Requirements
Python 3.x
Tkinter (included with most Python installations)
How to Run
Open a terminal or command prompt in the same directory as the program file.
Type python calculator.py and press Enter to run the program.
How to Use
Enter numbers using the keypad buttons.
Use the +, -, *, and / buttons to perform operations.
Press the Clear button to clear the input field.
Press the = button to calculate the result.
Known Issues
Division by zero is not handled and may result in an error.
